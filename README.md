# Winterfestival Statement

Official single-page Winterfestival statement site for Stichting Vlearmoesplein, built with Rust, Yew, WASM, and deployed via GitHub Pages.

## Tech Stack

- Rust (edition 2024)
- Yew (`csr`)
- Trunk (build/dev server)
- Tailwind CSS standalone + daisyUI plugin bundle
- GitHub Actions (GitHub Pages deployment)
- Renovate (dependency updates)

## Prerequisites

- Rust toolchain (`rustup`)
- `wasm32-unknown-unknown` target
- `trunk`
- `curl` (for downloading Tailwind standalone and daisyUI plugin files)

Quick setup:

```bash
make setup
```

This installs:

- Rust `wasm32-unknown-unknown` target
- `trunk` (if missing)
- local `tailwindcss` binary and `daisyui.mjs` plugin files (if missing)

If you only want to refresh CSS tooling:

```bash
make setup-css
```

## Local Development

Run dev server:

```bash
make serve
```

`make serve` automatically ensures Tailwind/daisyUI tooling exists before starting Trunk.
The CSS is generated by Trunk build hooks defined in `Trunk.toml` from `input.css`.

Build production assets:

```bash
make build
```

`make build` also auto-runs CSS tooling setup before compiling.

## daisyUI Integration

This project follows the Yew + daisyUI integration approach with:

- `input.css` (`@import "tailwindcss"` + `@plugin "./daisyui.mjs"`)
- `Trunk.toml` build hook generating `output.css`
- utility/component classes used directly in Yew markup

## Deployment (GitHub Pages)

The project includes `.github/workflows/pages.yml` to automatically deploy on pushes to `main`.
The workflow also downloads Tailwind standalone and daisyUI plugin files for Linux before running `trunk build`.

- For `username.github.io` repositories, the site is built with `/` as public URL.
- For project repositories (like `winterfestival-statement`), the site is built with `/winterfestival-statement/` as public URL.

In repository settings, set **Pages** source to **GitHub Actions**.

## Renovate

`renovate.json` is included with:

- `config:recommended`
- grouped updates for Cargo and GitHub Actions
- dependency dashboard enabled

Enable the Renovate GitHub App on this repository to activate automated update PRs.
